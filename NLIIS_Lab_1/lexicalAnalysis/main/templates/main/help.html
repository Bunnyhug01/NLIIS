Help

This application is designed for lexical, grammatical and syntax analysis of natural language.
The application has three modes of operation.

The first is text processing, it is designed to select lexemes from the text and output them with additional information (part of speech, prefixes, suffixes).
For text processing it is enough to write it manually or load it from a file and click the "Process" button.
All information will appear in the table.

All detected lexemes and their information is saved, so you can see their list in the "Saved Words" tab.
Also, this tab has the ability to search for a particular lexeme and if want you can delete some lexemes or the whole table.

The second mode is generation of word forms, based on the selected parameters (the choice of prefixes, suffixes).
To generate word forms, enter the word and press the "Generate" button. The whole list of word forms will appear in the table.

The third mode is syntax analysis, it is designed to build the syntax tree from the text.
For syntax analysis of the text it is enough to write it manually or load it from a file and click the "Analyze" button.

The fourth mode is semantic analysis, it is designed to build the semantic tree from the text.
For semantic analysis of the text it is enough to write it manually or load it from a file and click the "Analyze" button.

The last one is a "chat" that helps you get information about animals by sending messages.
